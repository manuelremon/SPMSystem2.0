                               RESUMEN RAPIDO
===============================================================================

PASO 2 (5 min): Desplegar Frontend en Vercel
  1. Abre: https://vercel.com/new
  2. Import repo: manuelremon/SPMSystem2.0
  3. Root: ./frontend, Build: npm run build, Output: dist
  4. Environment Variable: VITE_API_URL = https://spmsystem2-0.onrender.com/api
  5. Click Deploy
  6. COPIA la URL resultante (https://spmv2-0-xxxxx.vercel.app)
  
PASO 3 (1 min): Actualizar CORS en Render
  1. Abre: https://dashboard.render.com
  2. Servicio: spmsystem2-0 → Settings → Environment
  3. Edita FRONTEND_URL = https://tu-url-vercel.vercel.app
  4. Edita CORS_ORIGINS = https://tu-url-vercel.vercel.app,https://spmsystem2-0.onrender.com
  5. Save → Manual Deploy → Latest Commit

PASO 4 (2 min): Testing
  1. curl.exe https://spmsystem2-0.onrender.com/
  2. Abre https://tu-url-vercel.vercel.app en navegador
  3. Login: admin / a1
  4. Verifica dashboard
  5. (Opcional) python test_production.py
